Lederne

Lederne App

Lederne is a labor union for leaders, in Norway.

They wanted an app for their members to take advantage of the member-benefits they provide, such as articles, books, pod-casts, and courses/webinars.

For this project, I was the lead backend-developer for the cloud solution for the app.

The cloud solution included the following GCP products: Kubernetes Engine, Cloud Storage, Endpoints, Cloud Functions, Compute Engine, CloudSQL (postgres), Firestore, Firebase Remote Notifications, Firebase Auth, Stackdriver, Cloud DNS, Cloud NAT and Cloud Scheduler.

The project also consisted of a solid CI/CD pipeline in Cirlce CI.

The cloud solution provided a gRPC API for the apps to consume written in Golang, and Cloud Functions (JS) and ElasticSearch to provide searchable content.

March 2020 - Present
